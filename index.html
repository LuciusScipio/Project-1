üìä Project 3: Automate AWS Resource Monitoring with Python and CloudWatch

This project involved building an automated, serverless monitoring and alerting system for an AWS resource.

  • Tools Used: Python (Boto3), AWS CloudWatch, AWS SNS, AWS Lambda, AWS EventBridge
  • What I Learned: Programmatic monitoring with Python and Boto3, configuring CloudWatch alarms and dashboards, and using serverless functions for event-driven automation.

Steps I Followed:

  1. Set up a CloudWatch Dashboard to monitor EC2 instance CPU usage and an SNS topic for alerts.
  2. Wrote a Python script using boto3 to fetch CPU utilization data and publish messages to the SNS topic.
  3. Packaged the script and dependencies for deployment as an AWS Lambda function.
  4. Configured the Lambda function with the necessary IAM role permissions.
  5. Created a scheduled rule in EventBridge to trigger the Lambda function for automated monitoring.
